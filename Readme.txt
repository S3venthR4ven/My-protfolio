שם פרויקט: פורטפוליו
שם המגיש:  תור אחישר


תיאור קצר של הפרויקט:
דף פורטפוליו שנבנה לפי הנחיות ברורות עם שישה דוגמאות לדפי נחיתה / דפי אינטרנט. 
עמוד הפורטפוליו ושאר ששת העמודים נכתבו בשימוש של HTML, CSS, Bootstrap, SASS בהתאם להנחיות. 

הפרויקט נועד להראות את הבנתי מהמודול שעוסק ב-HTML ושאר דומיו שפות התגיות בעולם פיתוח האתרים תוך ניסיון בשימוש נכון בתגיות שלמדנו ורספונסיביות. 



סך הכול ניסיתי לכתוב את הקוד בצורה ברורה ומסודרת ככל האפשר. לגבי רספונסיביות טיפה התקשיתי (במיוחד עם bootstrap) ובניתי זאת תוך תשומת לב לכמה נקודות שבירה של העמוד (ולא רק לצורך התאמה להגדרת מסך רחב אחד והגדרת מסך טלפון אחת).
